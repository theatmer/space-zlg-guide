# space-zlg-guide

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本仓库主要存放若干独立开发的 HTML 页面，这些页面均为自包含的静态文件，不依赖外部服务器或数据库。每个页面均可在浏览器中直接打开运行，或部署至任何静态托管服务。

仓库本身不针对特定域名或网站，仅作为页面存档与展示的集合地。

## 目录说明

```
space-zlg-guide/
├── index.html          # 仓库总览页面（可选）
├── pages/              # 存放独立 HTML 页面的目录
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：核心目录，每个 `.html` 文件为一个独立页面。
- `assets/`：存放页面间可共用的静态资源，避免重复引用。

## 页面归档说明

- 所有页面均为纯前端实现，无后端交互。
- 页面内容、样式、逻辑均内嵌或引用仓库内资源。
- 每个页面通常包含一个独立的主题或功能，彼此不耦合。
- 页面可直接通过 `file://` 协议浏览，也可部署至任意 Web 服务器。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中打开 `pages/` 下的任意 HTML 文件即可查看。
3. 如需部署，将整个仓库上传至静态托管服务（如 GitHub Pages、Netlify 等），访问对应路径即可。
